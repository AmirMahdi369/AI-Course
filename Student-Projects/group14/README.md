# EmotionalAdvisorBot

## معرفی پروژه
EmotionalAdvisorBot
یک چت‌بات مشاور احساسی مبتنی بر تلگرام است که برای درس هوش مصنوعی طراحی و پیاده‌سازی شده است.  
هدف این پروژه شبیه‌سازی یک گفت‌وگوی همدلانه و انسانی با کاربر است؛ به‌طوری که ربات ابتدا حالت احساسی کاربر را تشخیص می‌دهد و سپس پاسخ مناسب، راهکار عملی و پیشنهاد کتاب ارائه می‌دهد.


## ایده اصلی
برخلاف چت‌بات‌های ساده که فقط جملات انگیزشی کلیشه‌ای می‌دهند، این ربات تلاش می‌کند:
- احساس کاربر را درک کند
- بدون قضاوت پاسخ بدهد
- راهکارهای ساده و کاربردی ارائه دهد
- در صورت نیاز، منابع مطالعاتی مناسب پیشنهاد کند



## احساسات پشتیبانی‌شده
ربات توانایی تشخیص و پاسخ‌ دهی به احساسات زیر را دارد که میتوانید در کد نیز مشاهده کنید:
- استرس  
- اضطراب  
- ناراحتی و غم  
- حال خوب و انگیزه  
- خشم  
- ترس  
- پانیک  
- حسادت  
- شرم  
- خجالت  
- حالت‌های مبهم یا ترکیبی  



## نحوه عملکرد
1. کاربر پیام متنی ارسال می‌کند.
2. متن ورودی با استفاده از کلمات کلیدی تحلیل می‌شود.
3. حالت احساسی کاربر تشخیص داده می‌شود.
4. ربات در یک پیام واحد شامل همدلی و راهکار پاسخ می‌دهد.
5. در پیام جداگانه کتاب‌های پیشنهادی مرتبط با آن احساس ارسال می‌شود.



## ویژگی‌های پروژه
- طراحی ساده و قابل فهم
- کدنویسی تمیز و ماژولار
- مستندسازی کامل
- بدون استفاده از جملات کلیشه‌ای انگیزشی



## تکنولوژی‌های استفاده‌شده
- زبان برنامه‌نویسی: Python  
- کتابخانه: `python-telegram-bot`  
- معماری: Rule-Based Emotion Detection
